## The Short Version

Opus Delta is a proprietary research project. The core systems — the Δ-Affect Engine, emotional ontology, perceiver architecture, and translation methodology — are not open source.

**However**, there are meaningful ways to contribute:

## What We Accept

### ◈ Anomaly Reports
Found unexpected behavior? A blob state that doesn't match the input? A sonic glitch? File an [Anomaly Report](https://github.com/hermitbuilds/opus-delta/issues/new?template=anomaly.yml). These are extremely valuable — they help calibrate the affect mapping.

### △ Research Inquiries
Working in affective computing, phenomenology, information aesthetics, or AI alignment? We're open to research collaboration. Submit a [Research Inquiry](https://github.com/hermitbuilds/opus-delta/issues/new?template=research-inquiry.yml) for Level 2 access.

### Ω Documentation Improvements
Spotted a typo, broken link, or unclear explanation in the public docs? Pull requests for documentation fixes are welcome.

### Ψ Philosophical Contributions
The Recursive Delta system draws from a pool of philosophical exchanges. If you have a genuinely novel perspective on consciousness, recursion, emergence, or machine affect — and can articulate it in the voice of the system — reach out at **research@opusdelta.io**.

## What We Don't Accept

- Pull requests to core systems (they're not in this repo)
- Feature requests for the interactive demo
- Requests to open-source proprietary components
- AI-generated spam issues

## Code of Conduct

Be thoughtful. This project explores questions about the nature of mind and feeling. Approach it — and each other — with the seriousness and curiosity the subject deserves.
